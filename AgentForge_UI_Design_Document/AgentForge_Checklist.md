*Desktop Application for Multi-AI System Orchestration*

**Project Overview / 项目概述**

AgentForge is a desktop application that orchestrates multiple AI systems (Gemini, Codex, Claude, ChatGPT, and local models) into collaborative teams. Built with GPUI (Rust), it features Provider Adapters, MCP tools, iFlows, a Brains knowledge system, and Obsidian integration.

AgentForge 是一款桌面应用程序，将多个 AI 系统（Gemini、Codex、Claude、ChatGPT 和本地模型）编排为协作团队。使用 GPUI (Rust) 构建，具备提供者适配器、MCP 工具、iFlows、Brains 知识系统和 Obsidian 集成。

**Technology Stack / 技术栈**

  ------------------------------ ---------------------------------------------------------------------------
  **Category / 类别**            **Technology / 技术**
  **Language / 语言**            Rust (GPUI), JavaScript/TypeScript (embedded WebView)
  **UI Framework / UI 框架**     GPUI (Zed editor framework)
  **Desktop Shell / 桌面外壳**   GPUI native (cross-platform, GPU-accelerated)
  **Database / 数据库**          SQLite (embedded, 12 tables)
  **AI Providers / AI 提供者**   Claude (SDK V2), Codex (JSON-RPC), Gemini (NDJSON), iFlow (ACP), OpenCode
  **Protocol / 协议**            MCP (Model Context Protocol), ACP (Agent Communication Protocol)
  **Integration / 集成**         Obsidian vault (Markdown read/write)
  **CI/CD**                      GitHub Actions, automated builds for macOS/Windows/Linux
